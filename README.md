## Clone the Repository
To get started, clone the repository using the following command:
```
git clone https://github.com/GirishTl/middle-east-localization.git
```

## Add Skills
Run the following command to add skills to your project:
```
 copilot add --skill ./
```

## Reload Skills
After adding the skills, reload them with:
```
copilot reload
```

## Verify Installation
Ensure everything is set up correctly by running:
```
 copilot verify
```
